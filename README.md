# Java Basics

This repository contains Java basic programs that cover core concepts required for backend and software engineering roles.

## Topics Covered
- Variables & Data Types
- Conditional Statements
- Loops
- Methods
- OOP Concepts
- Arrays & Collections

## Purpose
To build strong Java fundamentals and document my learning journey.


## Folder Structure
Each folder represents a Java concept with simple examples.

## How to Run?
1. Clone the repository
2. Compile using `javac`
3. Run using `java ClassName`
